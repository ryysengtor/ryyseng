Webhook & Feed API
Feed API publik tersedia di /api/feed — menampilkan transaksi sukses terbaru (tanpa data sensitif).
Cocok untuk testimoni di website lain atau dihubungkan ke bot WA.

Webhook dikirim otomatis ke URL kamu tiap ada order selesai, deposit masuk, atau sewa bot lunas. Header X-Ryysengtor-Secret bisa dipakai untuk verifikasi di sisi penerima.
Endpoint Publik
GET /api/feed — JSON, last 50 sukses
GET /api/feed/stream — SSE real-time
Query: ?type=order|deposit|sewabot&limit=20
Server akan POST ke sini tiap ada event sukses. Kosongkan untuk nonaktifkan.
Dikirim di header X-Ryysengtor-Secret.

Biaya Admin Deposit OTP (Nokos)
Biaya admin yang ditambahkan ke setiap deposit OTP. Pembeli melihat: Nominal deposit + biaya admin.
Contoh: user mau isi Rp10.000 → total yang dibayar = Rp10.000 + biaya admin. Saldo yang masuk ke akun mereka tetap Rp10.000.
Markup Harga Order OTP
Markup nominal (Rp) yang ditambahkan ke harga setiap order OTP dari RumahOTP.
Contoh: harga RumahOTP Rp500 + markup Rp200 = user melihat & membayar Rp700. Selisih Rp200 adalah keuntungan kamu.
Perubahan berlaku real-time tanpa perlu restart server.
Isi 0 untuk tidak menambah biaya. Tidak bisa diatur per-persentase.
Feed Cache Management
Feed cache menyimpan 50 transaksi terakhir di GitHub agar tetap muncul di /api/feed saat server restart (Vercel serverless).
Gunakan Lihat Cache untuk memverifikasi isi, atau Reset Cache jika ingin memulai dari nol.
